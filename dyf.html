DO YOU FEED?

NODE.JS AND EXPRESS WEB APP

Do You Feed focuses on analysing player metrics based on in game performance.

This was a side project I made to learn more about the MEAN stack. This application hits Riot Game's API endpoints to read player performance numbers and calculates appropriate metrics. It uses these metrics to generate satire related to how well they are performing in game.

  • Responsive Layout

    Using Foundations framework, the layout of this site is responsive to different form factors.

  • Custom Algorithms

    Created own algorithms to calculate player metrics and generate content within algorithm scope.

  • Stack Development

    Developed and designed user interface as well as back end endpoints.

  • NodeJS and Express

    Running javascript in NodeJS environment and using Express web framework for API.
