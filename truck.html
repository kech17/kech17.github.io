SENSOR ANALYSIS

CONTROLLERS AND CLOUD

Using sensors, radio modules, and an Arduino to update data to a cloud server in real-time.

The aim of the project was to gather data readings from sensors attached to a remote controlled truck. The sensors will detect behavioural trends in the truck such as acceleration, temperature, and rotation. This data is then communicated via wifi module to a cloud server which hosts a dashboard that displays metrics.

  • Arduino Microcontroller

    An Arduino Fio V3 microcontroller was used gather sensor data on the truck. C++ was used for code.

  • Sensors and Actuators

    Sensors allow environmental measurements and LEDs server to display behavioural processes of the code.

  • Cloud Computing

    A wifi module posts to a cloud server that analysizes and translates the raw sensor values.

  • Metrics and Analytics

    The cloud server hosts a dashboard page visualizing useful metrics and data trends from sensor values.
