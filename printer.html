3D PRINTER

REPRAP AND MODELLING

3D printing with the RepRap Mendel through calibration, software configuration, and construction.

The aim of this project is to work with 3D printers. Large modifications to the wiring, axis calibration, and software configuration was needed to start printing. Lots of documentation can be found online for the process of completing the 3D printer.

  • Wiring

    Wiring from the Arduino controller to the stepper motors required careful design and planning.

  • Software Configuration

    Open source software was needed to convert STL files to G-code for the 3D printer.

  • Calibration

    Both the XYZ axis and the extruder was calibrated for accurate printing.

  • Construction

    Learning the architectural design of the printer for reconstructing certain components.
